# Skill: image-gen

图片生成技能 - 基于 AI 生成高质量图片

## 功能

- 文生图：根据文字描述生成图片
- 支持多种分辨率和宽高比
- 自动处理中文提示词

## 使用口令

直接对我说：

- "生成一张猫咪的图片"
- "生成一张日落风景图"
- "生成一张未来城市的图片"

## 参数说明

| 参数 | 默认值 | 可选值 |
|------|--------|--------|
| 分辨率 | 2K | 1K, 2K, 4K |
| 宽高比 | 16:9 | 1:1, 2:3, 3:2, 3:4, 4:3, 4:5, 5:4, 9:16, 16:9, 21:9 |

## 注意事项

- 使用英文提示词效果更好
- 图片默认保存到当前工作目录
- 需要安装 Python 3.10+ 和 requests、Pillow 库

## 脚本文件

- `image_gen.py` - 独立图片生成脚本

## 使用示例

```bash
python image_gen.py "a cute cat"
python image_gen.py "sunset landscape" output.png
```
