# Skill: douyin-auto-browse

抖音自动浏览技能 - 自动滚动浏览抖音推荐视频

## 功能

- 自动打开抖音推荐页面
- **自动检测并跳过直播**
- 普通视频停留指定时间后自动切换
- 支持自定义停留时间、浏览数量
- 支持选择不同页面（推荐/精选/关注等）

## 使用方法

直接对我说：

- "启动抖音自动浏览" - 使用默认设置开始浏览
- "抖音浏览，每个视频停留10秒" - 自定义停留时间
- "抖音浏览20个视频" - 指定浏览数量
- "抖音浏览精选页面" - 浏览精选页
- "停止抖音浏览" - 停止当前浏览任务

## 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| interval | 10 | 每个视频停留时间（秒） |
| count | 50 | 浏览视频数量 |
| page | recommend | 页面类型：recommend(推荐)/jingxuan(精选)/follow(关注) |

## 直播检测逻辑

- 检查页面是否显示视频时长（如 `00:09 / 11:01`）
- 有时长显示 → 普通视频 → 停留指定时间
- 无时长显示 → 直播 → 自动跳过

## 依赖

- dumate-browser-use 技能（浏览器自动化）
